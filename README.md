TCS CodeVita Round 1 Z2
Description

This repository contains solutions in C++ for problems from the CodeVita Round 1 – Zone 2 contest hosted by Tata Consultancy Services (TCS). Each source file corresponds to a specific problem (e.g., abc.cpp, baker.cpp, ladder.cpp, smallest.cpp, text.cpp).

Directory Structure
/
├── abc.cpp
├── baker.cpp
├── ladder.cpp
├── smallest.cpp
└── text.cpp

Usage

Clone the repository

git clone https://github.com/EsDeath-Boop/TCS-CodeVita-R1Z2.git


Navigate into the folder

cd TCS-CodeVita-R1Z2


Compile a solution file. For example:

g++ -std=c++17 abc.cpp -o abc


Run the compiled executable and provide input as required by the problem.

./abc < input.txt
